# EnglishAPP v2 — 商務英文「對話流暢度」訓練

我自己學英文用的單檔 Web App（v2 重寫版）。目標：**口語超快速流暢**。

- 8 個商務情境的**分支劇本模擬對話**：TTS 扮演對方講話，你必須限時開口回應，語音辨識判斷你念了哪句、對話沿分支走下去。
- 三項流暢度評分：**反應時間**（對方講完到你開口）、**準確率**（逐字比對）、**語速**（words/sec）。
- 附**跟讀加速模式**（0.9x → 1.0x → 1.15x）練口腔速度。
- 單一 `index.html`，無外部相依、不接任何 API，部署 GitHub Pages，桌機 Chrome 使用。
- 進度（分數、連續天數）存 localStorage。
